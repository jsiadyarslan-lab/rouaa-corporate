# 📜 System Principles

> هذه الوثيقة مقدسة. كل قرار تقني، كل سطر كود، كل ميزة جديدة يجب أن يحترم هذه المبادئ.
> إذا تعارض أي تطوير مع هذه المبادئ، يُرفض التطوير.

## Principle 1: Source First، وليس News First
النظام لا يبدأ من الخبر. يبدأ من المصدر.
✅ مصدر → وثيقة → حدث → خبر (وليس العكس)

## Principle 2: المصادر الرسمية فقط
لا نعتمد على أي موقع إخباري. فقط: بنوك مركزية، هيئات إحصاء، جهات تنظيمية، بورصات، وزارات، هيئات دولية، صفحات IR للشركات.

## Principle 3: كل خبر ينتج من Event
لا يوجد خبر بدون Event. الـ Event هو الكيان الأول.

## Principle 4: كل تحليل مرتبط بـ Evidence
كل ادعاء يجب أن يرتبط بـ Document + Source + Hash.

## Principle 5: لا يوجد AI بدون دليل
الـ AI لا يهلوس. إذا لم يجد دليلاً، يقول "لا أعرف".

## Principle 6: كل Document قابل للتتبع
الوثائق محفوظة بنسخها الأصلية + hash + snapshot.

## Principle 7: البنية التحتية قبل الواجهة
نبني المحرك قبل السيارة. Data → Engine → API → UI.

## Principle 8: التزام بالترتيب
لا تخطي المراحل. لا Event قبل Document. لا Newsroom قبل Evidence.

## Principle 9: كل شيء قابل للبحث التاريخي
النظام يحفظ التاريخ. كل قرار، إيداع، تقرير، خطاب.

## Principle 10: التعدد اللغوي من اليوم الأول
كل Source، Document، Event، NewsItem له locale.
