# ADR-0003: Evidence-First Architecture

## Status
Accepted (2026-07-16)

## Context
الـ AI يهلوس لأنه لا يملك قيوداً. كل تحليل بدون دليل = احتمال خطأ.

## Decision
كل ادعاء (Claim) في النظام يجب أن يرتبط بـ Evidence:
```
Claim → Evidence → Document → Source
```

الـ AI لا يكتب شيئاً بدون دليل. إذا لم يجد دليلاً، يقول "لا أعرف".

## Consequences
- ✅ لا هلوسة
- ✅ قابلية تحقق كاملة
- ✅ ثقة المستخدم
- ⚠️ أبطأ في الإنتاج (يجب جمع الأدلة أولاً)
- ⚠️ بعض التحليلات الإبداعية قد تُرفض
