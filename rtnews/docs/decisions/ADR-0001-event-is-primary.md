# ADR-0001: Event is the Primary Entity

## Status
Accepted (2026-07-16)

## Context
في الأنظمة الإخبارية التقليدية، الخبر هو الكيان الأساسي. هذا يؤدي إلى:
- أخبار بدون مصدر موثق
- صعوبة تتبع الحقائق
- تكرار المحتوى
- هلوسة AI (لا يوجد مرجع)

## Decision
الـ **Financial Event** هو الكيان الأساسي في النظام.
الخبر (NewsItem) هو صياغة للحدث، وليس الحدث نفسه.

```
Event (primary) → Publication (derivative)
```

## Consequences
- ✅ كل خبر له مصدر موثق
- ✅ يمكن تتبع الحقائق
- ✅ منع التكرار (نفس الحدث = خبر واحد)
- ✅ AI لديه مرجع للالتزام به
- ⚠️ يتطلب بناء Event Engine قبل Newsroom
- ⚠️ بعض الأخبار "الناعمة" قد لا تناسب نموذج Event
