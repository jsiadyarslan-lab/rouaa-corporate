# 🧠 رؤى — Neural Core Redesign (Phase 1)

## تم إنشاء المرحلة الأولى: Hero Section + 3D Neural Sphere + Glassmorphism

### الملفات المُنشأة:
| الملف | الوصف |
|-------|-------|
| `index.html` | هيكل الصفحة مع RTL |
| `styles.css` | Glassmorphism + Glow + Animations |
| `main.js` | Three.js Neural Sphere + GSAP + Interactions |

### التأثيرات المُنفذة:
✅ **3D Neural Sphere** — كرة ذكاء اصطناعي 3D تتكون من 800+ نقطة مع خطوط عصبية متوهجة  
✅ **Glassmorphism Navigation** — شريط تنقل شفاف مع blur و glow  
✅ **Glow Text Effects** — عنوان "رؤى" بتأثير glow متدرج  
✅ **Magnetic Buttons** — أزرار تنجذب نحو الماوس  
✅ **3D Tilt Cards** — بطاقات الإحصائيات تميل باتجاه الماوس  
✅ **Animated Counters** — عداد أرقام متحرك عند التمرير  
✅ **Floating Ticker** — شريط أسعار متحرك بشكل loop  
✅ **Cursor Glow** — توهج يتبع الماوس (desktop only)  
✅ **Scroll Animations** — GSAP ScrollTrigger للأقسام  

### التقنيات المستخدمة:
- Three.js (WebGL 3D)
- GSAP + ScrollTrigger
- Custom Shaders (GLSL)
- CSS Glassmorphism
- CSS Animations

### كيفية التشغيل:
افتح `index.html` في المتصفح مباشرة (لا يحتاج server)

### للنشر على GitHub Pages:
1. ارفع الملفات الثلاثة إلى repository
2. اذهب إلى Settings → Pages
3. اختر Branch: main, Folder: root
4. انتظر الدقيقتين وستجد الموقع جاهزاً
